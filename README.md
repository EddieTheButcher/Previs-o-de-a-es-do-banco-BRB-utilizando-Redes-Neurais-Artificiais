# Previs-o-de-a-es-do-banco-BRB-utilizando-Redes-Neurais-Artificiais
Rede Neural LSTM feita em Python, utilizando os frameworks Keras e TensorFlow, para a previsão das ações BSLI3(ordinárias) e BSLI4(preferenciais ) do banco BRB. Os dados são públicos, retirados dos sites InfoMoney e Yahoo Finance.
